Understanding Mutable and Immutable Data Types in JavaScript

Immutable - Primitive Types

Immutable data types cannot be changed after they are created. When you update an immutable value, a new value is created.

  • String
  • Number
  • Boolean
  • Undefined
  • Null

Mutable - Non-Primitive Types

Mutable data types can be changed after they are created. When you update a mutable value, the original value is modified.

  • Array
  • Object
